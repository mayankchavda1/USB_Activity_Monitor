# USB Activity Monitor

A Python-based USB monitoring and forensic logging tool.

## Features

- USB insertion/removal detection
- File monitoring
- CSV/TXT logging
- Risk scoring
- Executable detection

## Technologies

- Python
- psutil
- watchdog
- colorama

## Run Project

```bash
python main.py
```
